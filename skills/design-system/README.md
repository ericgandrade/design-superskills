# Design System

Creates and maintains structured design systems using a three-layer token architecture (primitive, semantic, component), CSS variables, spacing and typography scales, and component state specifications. Enables consistent design-to-code handoff and brand-compliant presentation generation.

## Installation

```bash
npx claude-superskills install design-system
```

## Usage

Activate with any of these phrases:
- "create design tokens"
- "build design system"
- "define color tokens"
- "create component specs"
- "CSS variables"
- "spacing scale"
- "typography system"
- "brand-compliant presentation"

## Features

- Three-layer token architecture: primitive (raw values), semantic (purpose-mapped), component (context-specific)
- CSS custom properties generation for web and Tailwind theme configuration
- Spacing and typography scales with modular ratios and responsive breakpoints
- Component state specifications covering default, hover, active, focus, disabled, and error states
- Design-to-code handoff documentation with annotated component specs
- HTML slide generation with design tokens applied consistently across layouts
- Token migration utilities for evolving existing systems to structured architecture
