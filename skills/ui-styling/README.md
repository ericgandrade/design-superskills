# UI Styling

Implements beautiful, accessible user interfaces using shadcn/ui components (Radix UI + Tailwind CSS), utility-first CSS patterns, responsive layouts, dark mode, and canvas-based visual designs. Provides consistent, production-ready styling patterns for React-based frameworks and web applications.

## Installation

```bash
npx claude-superskills install ui-styling
```

## Usage

Activate with any of these phrases:
- "style this component"
- "add dark mode"
- "create responsive layout"
- "shadcn UI component"
- "Tailwind CSS"
- "implement design tokens"
- "accessible form"
- "canvas poster design"

## Features

- shadcn/ui component library with pre-built Radix UI primitives and Tailwind CSS styling
- Utility-first CSS patterns with Tailwind for rapid, consistent UI construction
- Responsive layouts using CSS Grid, Flexbox, and Tailwind breakpoint utilities
- Accessible components meeting WCAG 2.1 AA standards (dialogs, forms, tables, navigation)
- Dark mode implementation via CSS variables and Tailwind dark: modifier
- Canvas-based visual designs for posters, social cards, and decorative graphics
- Design token integration mapping brand variables to component-level CSS properties
- Consistent styling patterns across Next.js, Remix, Vite, and Astro projects
