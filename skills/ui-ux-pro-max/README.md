# UI/UX Pro Max

Comprehensive design intelligence for web and mobile projects, combining 50+ visual styles, 161 curated color palettes, 57 font pairings, and 99 UX guidelines across 10 technology stacks. Provides searchable CSV databases and Python search scripts for instant, data-driven design decisions.

## Installation

```bash
npx claude-superskills install ui-ux-pro-max
```

## Usage

Activate with any of these phrases:
- "design this UI"
- "build a landing page"
- "choose color palette"
- "design system for my app"
- "UX review"
- "implement this design"
- "what style should I use"
- "responsive layout"

## Features

- 50+ visual styles with opinionated stack-specific recommendations
- 161 color palettes with primary, secondary, accent, background, and text values
- 57 font pairings mapped to product types and brand personalities
- 10 technology stacks: React, Next.js, Vue, Svelte, Angular, Astro, HTML-Tailwind, shadcn/ui, Flutter, SwiftUI
- Searchable CSV databases with Python search scripts for palettes, fonts, and styles
- 99 UX guidelines covering interaction patterns, accessibility, and information architecture
- 25 chart types with data visualization recommendations per stack
- 161 product type profiles with reasoning rules for design decisions
