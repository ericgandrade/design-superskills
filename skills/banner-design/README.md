# Banner Design

Designs banners across 22 art direction styles (minimalist, gradient, bold typography, glassmorphism, neon, retro, geometric, and more) for social media, advertising networks, website heroes, and print formats. Generates multiple creative options per request with AI-powered visuals via Gemini and produces platform-specific dimensions automatically.

## Installation

```bash
npx claude-superskills install banner-design
```

## Usage

Activate with any of these phrases:
- "design a banner"
- "create social media banner"
- "Facebook ad banner"
- "LinkedIn banner"
- "YouTube banner"
- "create banner ad"
- "Google Display banner"
- "design website hero"

## Features

- 22 art direction styles from minimalist and corporate to neon, retro, and glassmorphism
- Social media banners for Facebook, Twitter/X, LinkedIn, YouTube, Instagram, and Pinterest
- Ad network formats: Google Display, Facebook Ads, LinkedIn Ads with standard IAB dimensions
- Website hero sections with responsive breakpoint variants
- Print banner formats for events, trade shows, and outdoor advertising
- AI-generated visual elements via Gemini for backgrounds, textures, and illustrations
- Multi-option output: generates 3–5 creative directions per request for client review
