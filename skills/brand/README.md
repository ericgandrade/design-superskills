# Brand

Defines and maintains brand identity systems including voice guidelines, visual identity standards, messaging frameworks, and asset management. Conducts brand consistency reviews, creates comprehensive style guides, and manages color palettes, typography specifications, and logo usage rules.

## Installation

```bash
npx claude-superskills install brand
```

## Usage

Activate with any of these phrases:
- "define brand voice"
- "create style guide"
- "brand guidelines"
- "tone of voice"
- "brand audit"
- "asset management"
- "brand consistency"
- "visual identity"

## Features

- Brand voice definition with personality traits, tone descriptors, and do/don't content examples
- Visual identity standards covering logo usage, clear space, incorrect usage, and co-branding rules
- Messaging framework creation with positioning statements, value propositions, and tagline options
- Brand consistency review and audit against established guidelines
- Asset management structure with naming conventions, approval workflows, and version control
- Color palette management with accessible contrast ratios and usage guidance
- Typography specification with hierarchy, weight mapping, and web/print rendering notes
- Complete style guide assembly combining all brand elements into a shareable reference document
