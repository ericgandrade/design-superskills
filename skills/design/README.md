# Design

Unified design skill covering brand identity, logo generation (55 styles with Gemini AI), corporate identity programs (50 deliverables with CIP mockups), HTML presentations, banner design (22 styles), icon design (15 styles SVG), and social media photo generation. A single entry point for all creative and visual production needs.

## Installation

```bash
npx claude-superskills install design
```

## Usage

Activate with any of these phrases:
- "design a logo"
- "create brand identity"
- "design banner"
- "create icon"
- "generate social media images"
- "corporate identity"
- "create mockups"
- "design slides"

## Features

- Logo generation in 55 styles powered by Gemini AI with SVG and PNG export
- Corporate identity programs with 50 deliverables and CIP mockup previews
- HTML presentations using Chart.js with responsive layouts and design tokens
- Banner design across 22 art direction styles for social, ads, web, and print
- Icon design in 15 SVG styles with scalable vector output
- Social media photo generation (HTML→screenshot) for all major platforms
- Brand style guide creation with color palettes, typography, and asset usage rules
