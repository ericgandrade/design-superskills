# Slides

Creates strategic HTML presentations with Chart.js data visualizations, design tokens, responsive slide layouts, and copywriting formulas (AIDA, PAS, STAR, and more). Applies contextual slide strategies for pitches, reports, education, and executive briefings with speaker notes.

## Installation

```bash
npx claude-superskills install slides
```

## Usage

Activate with any of these phrases:
- "create presentation"
- "build slides"
- "HTML slideshow"
- "pitch deck"
- "data visualization slides"
- "create slide deck"
- "generate presentation"
- "presentation with charts"

## Features

- Chart.js integration for bar, line, pie, scatter, radar, and mixed chart types
- Design token system for consistent colors, typography, and spacing across all slides
- Responsive HTML/CSS layouts that render correctly in browsers and can be exported
- Copywriting formulas applied per slide type: AIDA for sales, PAS for problem framing, STAR for storytelling
- Contextual slide strategy selection based on audience (executive, technical, sales, educational)
- Speaker notes generation aligned with slide content and presentation context
- Multiple layout types: cover, agenda, full-bleed, two-column, stat callout, closing
